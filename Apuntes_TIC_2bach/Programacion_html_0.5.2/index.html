Modales

Esto es un modal, para escribir uno tienes antes que crear un boton o enlace que lo "invoque", añadiendo data-toggle="modal" y data-target="#nombredelmodal" o href="#nombredelmodal".

Ahora para crear un modal debes poner lo siguiente:

<div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="myModalLabel">
  <div class="modal-dialog" role="document">
    <div class="modal-content">
      <div class="modal-header">
        <button type="button" class="close" data-dismiss="modal" aria-label="Close"><span aria-hidden="true">×</span></button>
        <h4 class="modal-title" id="myModalLabel">Modal title</h4>
      </div>
      <div class="modal-body">
        Texto de modal
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-default" data-dismiss="modal">Close</button>
        <button type="button" class="btn btn-primary">Save changes</button>
      </div>
    </div>
  </div>
</div>

Antes de programar

Antes de programar tienes que "crear" la página con unos ajustes para poder usar todo su potencial.

  1. Crea la cárpeta donde estarán todas las páginas de tu página web.
  2. Debemos descargar bootstrap y extraerlo en la carpeta creada.
  3. Descargamos jquery y metemos el archivo en la carpeta js (creada previamente por bootstrap).
  4. Creamos carpeta img en la carpeta de la web, donde guardaremos las imagenes usadas.

Una vez hecho todo esto, descargamos "CoffeeCup Free", el editor de html y podremos programar html, pero para que utilicemos lo previamente instalado tenemos que poner lo siguiente en todos los códigos:

<link href="css/bootstrap.min.css" rel="stylesheet">
(Antes de style, porque estamos abriendo la carpeta css) 
<script type="text/javascript" src="js/jquery.js"></script>
<script src="js/bootstrap.min.js"></script>

El jquery debe ir antes de que pongamos la etiqueta script (la cual no he usado en esta web, la explico en mi web de javascript)
