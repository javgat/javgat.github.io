Tablas

Las tablas sirven para hacer tablas. Se inicia con etiqueta <table>, crear filas con <tr> y luego celdas en cada fila <td>. Para celda "título" (centrada en negrita) se usa <th> Un ejemplo de tabla y luego su representación: 

<table>
	<tr>
		<th>Titulo Columna 1</th>
		<th>Título Columna 2</th>
	</tr>
	<tr>
		<td>Celda 2,1</td>
		<td>Esto es una celda</td>
	</tr>
	<tr>
		<td>Celda 3,1</td>
		<td>Celda 3,2</td>
	</tr>
</table>
(Para escribir el codigo directamente como aquí, he puesto <pre></pre> )

Clases de tablas Bootstrap

Bootstrap vuelve a salvarnos el día y nos permite modificar la tabla para tener líneas y diseños "de fábrica" que no den asco. Añadiendo class="table" se ve así:

Titulo Columna 1 Título Columna 2
Celda 2,1 Esto es una celda
Celda 3,1 Celda 3,2

Las clases disponibles son:

  • table
  • table-striped
  • table-bordered
  • table-hover
  • table-condensed

Pero por si esto no fuera suficiente, bootstrap nos brinda la posibilidad de modificar las columnas (tr) o las celdas con las siguientes clases:

  • active
  • success
  • warning
  • danger
  • info

Modales

Esto es un modal, para escribir uno tienes antes que crear un boton o enlace que lo "invoque", añadiendo data-toggle="modal" y data-target="#nombredelmodal" o href="#nombredelmodal".

Ahora para crear un modal debes poner lo siguiente:

<div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="myModalLabel">
  <div class="modal-dialog" role="document">
    <div class="modal-content">
      <div class="modal-header">
        <button type="button" class="close" data-dismiss="modal" aria-label="Close"><span aria-hidden="true">×</span></button>
        <h4 class="modal-title" id="myModalLabel">Modal title</h4>
      </div>
      <div class="modal-body">
        Texto de modal
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-default" data-dismiss="modal">Close</button>
        <button type="button" class="btn btn-primary">Save changes</button>
      </div>
    </div>
  </div>
</div>
